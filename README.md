# AI Calculator - Word Problem Solver

A powerful web-based calculator with AI-powered word problem solving capabilities. Hosted on GitHub Pages!

🔗 **[Live Demo](https://dandziewit.github.io/AI-Calculator/)** - Try it now!

![Calculator Screenshot](screenshot.png) *(Optional: Add a screenshot)*

## ✨ Features

### 🧮 Standard Calculator
- Basic arithmetic operations (+, −, ×, ÷)
- Parentheses support
- Clean, modern interface

### 🤖 AI Word Problem Solver
Understands natural language math questions and provides step-by-step explanations:

- **Percentage Problems**: Discounts, taxes, tips, "X% of Y"
- **Rate Problems**: Wages, costs per item, speed calculations
- **Comparison Problems**: Twice, half, more than, less than
- **Multi-Step Problems**: Sequential calculations
- **Averages**: Mean calculations
- **Simple Equations**: Linear equation solving
- **Arithmetic**: Basic operations in natural language

### 📜 Additional Features
- Calculation history tracking
- Step-by-step explanations
- Confidence scoring for AI interpretations
- Clean, responsive design
- Dark mode interface

## 🚀 Deployment to GitHub Pages

### Method 1: Using GitHub Web Interface

1. **Push to GitHub**:
   ```bash
   git add index.html style.css calculator.js README.md
   git commit -m "Add web calculator"
   git push origin main
   ```

2. **Enable GitHub Pages**:
   - Go to your repository on GitHub
   - Click **Settings** → **Pages**
   - Under "Source", select **main** branch
   - Click **Save**
   - Your site will be live at: `https://dandziewit.github.io/Rectangle/`

### Method 2: Using GitHub Desktop

1. Open GitHub Desktop
2. Select your changes
3. Commit with message: "Add web calculator"
4. Push to origin
5. Follow step 2 from Method 1

## 💻 Local Development

To run locally:

1. Clone the repository:
   ```bash
   git clone https://github.com/dandziewit/Rectangle.git
   cd Rectangle
   ```

2. Open `index.html` in your browser, or use a local server:
   ```bash
   # Python 3
   python -m http.server 8000
   
   # Or use VS Code Live Server extension
   ```

3. Navigate to `http://localhost:8000`
